# Harmony Platform – System Overview

## Vision
Harmony is a Personal Destiny Platform that provides users with a persistent destiny profile and daily personalized insights based on Eastern astrology principles.

## Ecosystem Structure

VUTERA (Company)
    └── Harmony (Spiritual Platform)
            ├── TuVi (Traffic & SEO Engine)
            └── AnMenh (User Identity & Core Product)

## Core Flow

Visitor → TuVi Article → CTA → Create Destiny Profile → AnMenh Dashboard → Daily Return → Monetization

## Core Principles

1. TuVi does NOT require login.
2. AnMenh requires account.
3. All personalization happens in AnMenh.
4. TuVi exists to acquire users.
5. AnMenh exists to retain and monetize users.
